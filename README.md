# אתר תדמית — עבדו קבב 🔥

אתר תדמית לעמוד אחד (single-page) למסעדת בשרים על האש. בנוי ב-HTML / CSS / JavaScript טהור, ללא תלות בחבילות וללא שלב build. עברית מלאה, RTL, ורספונסיבי למובייל.

## מבנה הפרויקט

```
AbduKabab/
├── index.html        # מבנה העמוד וכל הסקשנים
├── css/styles.css    # כל העיצוב (צבעים דרך משתני CSS ב-:root)
├── js/
│   ├── data.js       # ⭐ כל התוכן — ערוך כאן: פרטים, תפריט, שעות, ביקורות, מפה
│   ├── main.js       # לוגיקה: ניווט, תפריט, גלריה, lightbox, שעות, טופס
│   └── reviews.js    # ביקורות גוגל (חיות או סטטיות)
├── images/           # תמונות (כרגע placeholder מ-Unsplash)
└── README.md
```

## הרצה

פשוט פותחים את `index.html` בדפדפן (דאבל-קליק).

> כדי שביקורות Google API יעבדו צריך להריץ דרך שרת (לא `file://`). הדרך הקלה:
> ```powershell
> # מתוך תיקיית הפרויקט
> python -m http.server 8000
> ```
> ואז גולשים אל http://localhost:8000

## איך מעדכנים תוכן

**כמעט הכל נמצא בקובץ אחד: `js/data.js`.**

| מה לעדכן | היכן |
|---|---|
| טלפון, אימייל, כתובת, כשרות | `SITE_CONFIG` בראש `data.js` |
| רשתות חברתיות (אינסטגרם/פייסבוק/טיקטוק) | `SITE_CONFIG` |
| הסיפור ("מי אנחנו") | מערך `STORY.paragraphs` |
| מחירון שיפודים | `MENU_SKEWERS` (מטריצת מחירים) |
| ספיישלים | מערך `MENU_SPECIALS` |
| תוספות ושתייה | מערך `MENU_EXTRAS` |
| תמונות גלריה | מערך `GALLERY` |
| שעות פתיחה | מערך `HOURS` |
| ביקורות גוגל (סטטי) | מערך `STATIC_REVIEWS` |
| מיקום למפה ולניווט | `SITE_CONFIG.mapQuery` |

> שים לב: מספר הטלפון מופיע גם ב-`index.html` בקישורי `tel:`.
> חפש שם `97249988555` / `04-9988555` כדי להחליף.

### מבנה מחירון השיפודים (`MENU_SKEWERS`)
כל מנה מוגדרת כך — שלושה זוגות מחירים, לפי סדר ההגשה:
```js
{ name: "קבב", p: [[35, 50], [45, 55], [45, 65]] }
//                  פיתה      בגט/לאפה   צלחת/חמגשית
//                [שיפוד 1, 2 שיפודים] לכל אחד. null = לא זמין.
```

## מפה וניווט

- **כפתורי Waze / Google Maps** עובדים אוטומטית לפי `lat`/`lng` ב-`data.js`.
- **המפה המוטמעת** (iframe): ב-Google Maps מצא את המסעדה → *Share* → *Embed a map* → העתק את ה-`src` והחלף אותו בתגית ה-`<iframe id="mapIframe">` ב-`index.html`. לא דורש מפתח API.

## ביקורות מ-Google (אופציונלי — חי)

ברירת המחדל מציגה את `STATIC_REVIEWS` מ-`data.js`. לביקורות חיות:

1. ב-[Google Cloud Console](https://console.cloud.google.com/) צור פרויקט, הפעל **billing**, והפעל את **Places API** + **Maps JavaScript API**.
2. צור **API Key** והגבל אותו ל-*HTTP referrers* (הדומיין שלך) — חשוב, המפתח חשוף בצד הלקוח.
3. מצא את ה-**Place ID** של המסעדה: https://developers.google.com/maps/documentation/places/web-service/place-id
4. הזן ב-`data.js`:
   ```js
   googleApiKey: "YOUR_KEY",
   googlePlaceId: "YOUR_PLACE_ID",
   ```

מגבלות Google: עד **5 ביקורות**, ולא ניתן לבחור אילו. אם הקריאה נכשלת — האתר חוזר אוטומטית לביקורות הסטטיות.

> כרגע האתר מציג **ביקורות סטטיות אמיתיות** מתוך `STATIC_REVIEWS` ב-`data.js`. כדי להוסיף/לעדכן ביקורת, ערוך את המערך (שם, דירוג, טקסט, תאריך יחסי, ומידע נוסף ב-`meta`).

## יצירת קשר

אזור יצירת הקשר מציג כרטיסי טלפון, אימייל, כתובת ושעות, וכפתור חיוג בולט (אין טופס שליחה — הפנייה היא טלפונית/אימייל ישיר). רשתות חברתיות (אינסטגרם, פייסבוק, טיקטוק) מופיעות בתחתית האזור עם הלוגואים שלהן.

## עיצוב / צבעים

כל הצבעים מוגדרים כמשתני CSS בראש `css/styles.css` (`:root`). שינוי `--primary` / `--accent` מחליף את כל ערכת הצבעים.
