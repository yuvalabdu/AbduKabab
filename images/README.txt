תיקיית תמונות — עבדו קבב
==========================

כאן ניתן לשמור את התמונות של המסעדה (לוגו, מנות, אווירה).

כרגע האתר משתמש בתמונות לדוגמה מ-Unsplash (קישורים חיצוניים) כדי שייראה מלא.
כדי להשתמש בתמונות שלך:

1. שמור את התמונות בתיקייה הזו, למשל:
   images/kebab.jpg
   images/gallery-1.jpg
   images/about.jpg

2. ערוך את הקובץ js/data.js — מערך GALLERY — והחלף את ה-src בנתיב המקומי:
   { src: "images/gallery-1.jpg", alt: "תיאור התמונה" }

3. לתמונת ה-Hero ו-"מי אנחנו": ערוך את css/styles.css (חפש "unsplash")
   ואת index.html (תגית ה-img בסקשן about).

המלצות:
- רוחב מומלץ לתמונות גלריה: ~800px, ל-Hero: ~1920px.
- פורמט JPG/WEBP, דחיסה סבירה (איכות 80%) לטעינה מהירה.
